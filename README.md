# Netflix Landing Page Clone 🎬

This project is a frontend clone of the Netflix landing page, developed using pure HTML5 and CSS3.

The goal was to replicate a real-world streaming platform layout while maintaining clean structure, responsive behavior, and scalable styling practices.

---

## 🚀 Project Overview

This project focuses on:

- Full-width hero banner with background overlay
- Call-to-action section with input field
- Multiple structured content sections
- FAQ layout design
- Footer layout replication
- Responsive behavior across devices

The layout adapts smoothly across desktop, tablet, and mobile screens.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox)
- Media Queries for responsiveness

---

## 📱 Key Features

- Dark theme UI implementation
- Section-based layout structuring
- Proper spacing and alignment system
- Responsive navigation and content
- Clean and maintainable CSS structure
